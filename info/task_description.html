A give list should be "compressed" in a way so, instead of two (or more) equal elemets, staying one after another, there is only one in the result array.

Input: {% if interpreter.slug == "js-node" %}Array{% else %}Iterable{% endif %}.

Output: "Compressed" {% if interpreter.slug == "js-node" %}Array{% else %}Iterable{% endif %}.

Example:

{% if interpreter.slug == "js-node" %} 
conpress([5,
  5,
  5,
  4,
  5,
  6,
  6,
  5,
  5,
  7,
  8,
  0,
  0]) == [5, 4, 5, 6, 5, 7, 8, 0]
conpress([1, 1, 1, 1, 2, 2, 2, 1, 1, 1]) == [1, 2, 1]
{% else %} 
compress([5,
  5,
  5,
  4,
  5,
  6,
  6,
  5,
  5,
  7,
  8,
  0,
  0]) == [5, 4, 5, 6, 5, 7, 8, 0]
compress([1, 1, 1, 1, 2, 2, 2, 1, 1, 1]) == [1, 2, 1]
{% endif %}
